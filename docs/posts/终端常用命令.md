终端常用命令

```sh
1. **导航命令:**
   -  `cd` : 切换目录
   -  `ls` : 列出文件和目录
   -  `pwd` : 显示当前工作目录
   -  `mkdir` : 创建目录
   -  `rmdir` : 删除目录

2. **文件管理命令:**
   -  `touch` : 创建新文件
   -  `rm` : 删除文件
   -  `cp` : 复制文件或目录
   -  `mv` : 移动或重命名文件或目录

3. **查看和编辑文件:**
   -  `cat` : 显示文件内容
   -  `more`  或  `less` : 逐页查看文件内容
   -  `nano`  或  `vim` : 文本编辑器，用于编辑文件

4. **进程管理:**
   -  `ps` : 显示进程信息
   -  `kill` : 终止进程
   -  `top`  或  `htop` : 显示系统进程和资源使用情况

5. **网络命令:**
   -  `ping` : 检查网络连接
   -  `ifconfig`  或  `ip` : 显示网络接口信息
   -  `ssh` : 安全 Shell，用于连接远程服务器

6. **系统信息:**
   -  `uname` : 显示系统信息
   -  `df` : 显示磁盘空间使用情况
   -  `free` : 显示内存使用情况

这些是终端中常用的一些命令。如果您需要更多具体的信息或对任何命令有疑问，请随时告诉我。
```
